# Codex Workflow

1. Run `chub search <topic>` before scanning the whole repo.
2. Use `chub get <id>` to pull the exact document into agent context.
3. Add short notes with `chub annotate <id> "<note>"` after reviewing a file.
4. Use `chub stats` to confirm the registry stays small on low-memory hosts.

This sample skill file is plain Markdown so it can be indexed directly by `chub`.
